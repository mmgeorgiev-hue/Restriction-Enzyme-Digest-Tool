# Restriction-Enzyme-Digest-Tool
Run a restriction digest for iPCR prioritization. Upload a reference genome (in FASTA format), configure digest parameters, and get a ranked summary of restriction enzymes for iPCR experiments.


##Run locally
git clone https://github.com/mmgeorgiev-hue/Restriction-Enzyme-Digest-Tool.git
cd Restriction-Enzyme-Digest-Tool
python3 -m pip install -r requirements.txt
streamlit run app.py
